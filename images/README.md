# Images Folder

Place your image assets here:

## Required Images:
- `logo.png` - ZebraGo / BSCC logo (recommended size: 200x200px)

## Optional Images:
- Carousel background images (if you want custom backgrounds)
- Service icons
- Team photos
- Any other graphics

## Image Guidelines:
- Use optimized images (compress before uploading)
- Recommended formats: PNG for logos/icons, JPG for photos, SVG for graphics
- Keep file sizes reasonable for web performance
