# Logo Folder

Please place your logo images here:

- `zebrago-logo.png` - ZebraGo logo (left side)
- `bscc-logo.png` - BSCC/匡智會 logo (right side)

## Recommended Sizes:
- ZebraGo logo: Height ~200-250px (will be displayed at 70px height)
- BSCC logo: Height ~200-250px (will be displayed at 65px height)
- Use transparent PNG format for best results
- Logos will maintain aspect ratio automatically
