# Contact Page Icons

Please add the following icon images to this folder:

1. **location-icon.png** - Location/Address icon (24x24px recommended)
2. **phone-icon.png** - Phone icon (24x24px recommended)
3. **fax-icon.png** - Fax icon (24x24px recommended)
4. **email-icon.png** - Email icon (24x24px recommended)

## Specifications:
- **Size**: 24x24 pixels (or larger with transparent background)
- **Format**: PNG with transparent background
- **Color**: Primary green (#2D9CDB) or black (will adapt to design)
- **Style**: Simple, clean line icons or filled icons

## Usage:
These icons are used in the contact.html page to display contact information with visual indicators.
